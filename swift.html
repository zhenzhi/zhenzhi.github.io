SwiftKey Usability Testing

Summary

This usability testing is intended to evaluate gesture text entry performance on SwiftKey for iOS compared against text entry on the keyboard standard iOS keyboard. The usability test will record time on task, error rate, efficiency, learnability, and user satisfaction.
